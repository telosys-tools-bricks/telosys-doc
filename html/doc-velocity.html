Telosys templates language

The code generation is based on "templates"
A template is a file with a ".vm" extension written in "Velocity Template Language"

Velocity is a well known Open Source template engine ( cf http://velocity.apache.org/ )
The Velocity engine used in Telosys Tools is version 1.7

The main elements of the Velocity Templates Language (VTL) are documented below.

Velocity standard syntax and directives

  • #foreach
  • #if
  • #include
  • #macro
  • #parse
  • #set
  • comments
  • unparsed content

Telosys specific directives

  • #assertFalse
  • #assertTrue
  • #checkId
  • #error
  • #using
