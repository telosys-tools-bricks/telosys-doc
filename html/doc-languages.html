Telosys target languages

Telosys is able to generate any type of language.
However, for the most common languages it is possible to indicate the nature of the target language so as to simplify the writing of templates.

The target language can be specified in the template files (.vm) using the following directive :

    #set( $env.language = 'LanguageName' )

Thus Telosys will be able to apply certain default behaviors,
for example to convert the 'neutral type' of the model into a language-specific type
or to determine literal values adapted to the language.

Documentation for languages pre-configured in Telosys :

  • C# (Microsoft CSharp)
  • Go (GoLang)
  • Java
  • JavaScript
  • PHP
  • Python
  • TypeScript
