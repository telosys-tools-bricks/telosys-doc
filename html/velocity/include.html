#include

Imports a local file, which is then inserted into the location where the #include directive is defined
The file is included as is (as a text file, not rendered through the template engine)
The files to be included must be located in the Telosys Templates directory

Examples :

Include a single file :

    #include("header.include")
   
    #include("myfile.txt")
   
    #include("include/myfile.txt")


Include multiple files :

    #include("header.include" "myfile.txt")
   
    #include("header.include", "myfile.txt")
   
    ## The 2 files are concatenated and inserted

Include with variables :

    #include($file1)
   
    #include("header.include", $file2)
